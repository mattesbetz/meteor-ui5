Class: PropertyListBinding

.mongo. PropertyListBinding

The PropertyListBinding is a specific binding for lists in the model, which can be used to populate Tables or ItemLists.

getCurrentContexts()

Returns an array of currently used binding contexts of the bound control This method does not trigger any data requests from the backend or delta calculation, but just returns the context array as last requested by the control. This can be used by the application to get access to the data currently displayed by a list control.

getProperty(sPath [, oContext])

Returns the value of a property at a given path in a given context.

Resolves a context and path and returns the value (of any data type). This method also handles Lookup Property paths in the form of ?Customers("CustomerID") where ?Customers is the Mongo collection to be queried and "CustomerId" is the name of the property at the current path that contains the unique Mongo Id of the customer document.
Parameters:
Name Type Argument Default Description
sPath string the path to where to read the attribute value
oContext object <optional>

.mongo. DocumentListBinding

new DocumentListBinding(oModel, sPath, oContext [, aSorters] [, aFilters] [, mParameters])

Constructor for DocumentListBinding

Provides a specialized context binding that can be used to bind to several documents in a Meteor Mongo Collection. It is this type of ContextBinding that is used when, for example, an object header is bound to a path like "/Orders". The results can be restricted via the aFilters parameter. Each instance of this class observes changes on a query handle to provide reactive updates via firing events.
